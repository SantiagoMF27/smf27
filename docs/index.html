Acerca de mí

¿Qué es un Front end Developer?

Un Front end Developer es aquella persona encargada de programar el navegador de una web, es decir, es el encargado de traducir las definiciones de diseño y estilo visual realizadas en etapas previas a códigos HTML semántico. Es una de las piezas fundamentales en un proyecto digital.

Aunque normalmente a un Front end Developer se le asocien principios de diseño y de estructura de páginas web, como buen programador que es debe ser consciente que su trabajo se realizará en el lado cliente, en muchos casos en el navegador. Pero aún así, esta persona ha de tener conocimientos y tener en cuenta la usabilidad, legibilidad de la web o app, el diseño y la estructura de páginas web y aplicaciones para procurar que el usuario pueda tener una experiencia de 10.

Nuestros servicios

Nuestros servicios digitales

Diseño Web

Consiste en aplicar un conjunto de conocimientos de programación, psicología, diseño y comunicación con el fin de crear sitios web con las características y requisitos necesarios para representar un negocio, empresa o actividad en Internet.

Desarrollo Web

Desarrollo web significa construir y mantener sitios web; es el trabajo que tiene lugar en un segundo plano y que permite que una web tenga una apariencia impecable, un funcionamiento rápido y un buen desempeño para permitir la mejor experiencia de usuario.

Aplicación Android

Es un programa, un software diseñado para el sistema operativo Android que se instala fácilmente y cumple distintos objetivos según sus desarrolladores y el lenguaje de programación utilizado para su creación.

Fotografía

La fotografía es el arte y la técnica de obtener imágenes duraderas debido a la acción de la luz. ​Es el proceso de proyectar imágenes, capturarlas y plasmarlas, bien por medio del fijado en un medio sensible a la luz o por la conversión en señales electrónicas.

Redacción de Contenido

La redacción de contenido es el proceso de investigar, planificar, escribir, editar y publicar contenido para la web. Puede ser un post de blog, un guión para un vídeo, una página de ventas—cualquier cosa que se publique online.

Edición de Video

La edición de vídeo es el proceso que permite componer un producto audiovisual terminado y listo para su visionado a partir de fragmentos de vídeo, imágenes, sonido, subtítulos, efectos especiales o cualquier otro material grabado previamente.
